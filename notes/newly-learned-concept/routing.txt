| Snapshot                       | Observable              |
| ------------------------------ | ----------------------- |
| One-time value                 | Continuous updates      |
| Synchronous                    | Asynchronous            |
| No subscription                | Requires subscription   |
| Doesn't react to route changes | Reacts to route changes |
| Simple                         | More flexible           |

| navigate()              | navigateByUrl()     |
| ----------------------- | ------------------- |
| Array-based             | String-based        |
| Supports route segments | Uses full URL       |
| Supports `relativeTo`   | No `relativeTo`     |
| Easier for params       | Good for fixed URLs |
